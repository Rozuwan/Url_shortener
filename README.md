# URL Shortener

[![Live](https://img.shields.io/badge/Live-roxlink.vercel.app-0ea5e9?style=for-the-badge&logo=vercel&logoColor=white)](https://roxlink.vercel.app/)

A minimalist URL shortener built with React + Vite.

```mermaid
flowchart LR
  A[Long URL] --> B[Shorten]
  B --> C[Short Link]
  C --> D[Copy]
  D --> E[Done]
```

## Stack

**React 19 · Vite · Tailwind CSS 4 · Axios · Sonner · Three.js/Vanta**

## Quick Start

```bash
npm install
npm run dev
```

## Features

- Paste long URL → get short link
- One-click copy to clipboard
- Fog background effect
- Toast notifications
