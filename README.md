# URL Shortener

**Live:** [roxlink.vercel.app](https://roxlink.vercel.app/)

A minimalist URL shortener built with React + Vite.

```mermaid
flowchart LR
  A[Long URL] --> B[Shorten]
  B --> C[Short Link]
  C --> D[Copy]
  D --> E[Done]
```

## Stack

**React 19 · Vite · Tailwind CSS 4 · Axios · Sonner · Three.js/Vanta**

## Quick Start

```bash
npm install
npm run dev
```

## Features

- Paste long URL → get short link
- One-click copy to clipboard
- Fog background effect
- Toast notifications
